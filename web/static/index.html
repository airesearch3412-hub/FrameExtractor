      使用說明

      五大功能

      1. 🎬 提取 + 去重：從影片抽出畫面並自動去除重複幀。
      2. 📸 只提取：依「抽幀間隔」輸出畫面，不做去重。
      3. 🗂 僅去重資料夾：對既有圖片資料夾去重（移動 / 刪除 / 僅報表）。
      4. 📚 批次處理：一次處理多支影片，各自輸出子資料夾。
      5. ✂ 批次裁剪：以同一裁剪框批次裁切多張圖片。

      演算法等級

      • 快速：dHash（連續幀快篩）
      • 標準：dHash + pHash（推薦）
      • 精準：＋直方圖 + SSIM
      • 最精準：＋CLIP 語意（需 PyTorch，較慢）
